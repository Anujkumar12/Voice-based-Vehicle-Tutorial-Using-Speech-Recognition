Voice-based-Vehicle-Tutorial-Using-Speech-Recognition
Overall basic working of the project-
1.Execute the project from the Visual Studio
2.then on console you will see two basic operations
    -- Train
    -- Play
3.Train module is for new user ,so that system builds the model for the respective users voice i.e live testing.
4.Play is for old users,on procedding with play you need to speak name of the type of the vehicle
5.then it will show the respective image corresponding to that image. The image will close automatically after few seconds.
